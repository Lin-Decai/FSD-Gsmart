ROS编译配置说明
    切换ROS1指令：sh configure.sh ROS1
    切换ROS2指令：sh configure.sh ROS2

编译说明
	ROS1: catkin_make install --source ./src/pbox_node
	ROS2: colcon build --symlink-install

协议裁剪说明
	/src/pbox_node/protocol目录下的文件，对应BDDB协议的解析
	step1: 删除对应的协议文件
	step2: main.cpp中屏蔽对应的include 语句

ROS1运行指令：
    终端1：
        source /opt/ros/noetic/setup.bash          #留意ROS安装目录
        roscore
    新开终端2：
        source /opt/ros/noetic/setup.bash          #留意ROS安装目录
        source install/setup.bash                  #在pbox_node节点文件下
        roslanuch pbox_node pbox_node.launch
    新开终端3：
        source /opt/ros/noetic/setup.bash          #留意ROS安装目录
        source install/setup.bash                  #在pbox_node节点文件下
        rostopic list
        rostopic echo /pbox_pub/Gnss                #查看节点数据
        #如需保存csv
        rostopic echo /pbox_pub/Gnss --csv > gnss.csv

ROS2运行指令：
    终端1：
        source /opt/ros2/humble/setup.bash          #留意ROS2安装目录
        source install/setup.bash                  #在pbox_node节点文件下
        ros2 lanuch pbox_node pbox_node.launch
    新开终端3：
        source /opt/ros2/humble/setup.bash          #留意ROS安装目录
        source install/setup.bash                  #在pbox_node节点文件下
        ros2 topic list
        ros2 topic echo /Gnss                		#查看节点数据
        #如需保存csv
		ros2 topic echo --qos-depth 100000 /Gnss --csv > gnss.csv
